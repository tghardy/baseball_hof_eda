# What's in this repository?

**baseball_hof.csv**- Data scraped from [Wikipedia](https://en.wikipedia.org/wiki/List_of_members_of_the_National_Baseball_Hall_of_Fame) on the professional Baseball Hall of Fame. Contains the features player name, team, position, career span, and vote percentage.

**blog.md**- Written analysis of the data. [View this document on my website!](https://tghardy.github.io/github_pages/projects/data-acquisition.html)

**EDA.ipynb** Notebook containing code for EDA

**scraping.ipynb**- Code for obtaining the data
